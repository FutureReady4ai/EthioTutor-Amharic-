# አማርኛ ይማሩ — Amharic Learning Platform

A fully offline Amharic language learning application featuring the complete Fidel script, vocabulary lessons, interactive quizzes, and comprehensive grammar reference.

![Amharic Tutor](https://img.shields.io/badge/Language-Amharic-green)
![License](https://img.shields.io/badge/License-MIT-blue)
![Status](https://img.shields.io/badge/Status-Active-brightgreen)

## Features

✨ **Complete Fidel Script** — All 33 base consonants with 7 vowel variations each (230+ characters)
🔢 **Amharic Numerals** — Full set of Ethiopic numbers (1-100 and beyond)
📚 **5 Vocabulary Lessons** — Greetings, Numbers, Essentials, Food & Drink, Family (50+ phrases)
🎯 **Interactive Quiz** — 15 randomized questions covering vocabulary and grammar
📖 **Grammar Reference** — 11 comprehensive topics including word order, verbs, pronouns, negation
💻 **Fully Offline** — Works without internet after initial load
📱 **Mobile Friendly** — Responsive design for all devices
⚡ **Fast & Lightweight** — 46KB single HTML file, loads in milliseconds

## Quick Start

### Option 1: Visit Online
Open directly in your browser: **https://ethioputor-amharic.vercel.app**

### Option 2: Embed on Your Website
```html
<iframe 
  src="https://ethioputor-amharic.vercel.app" 
  width="100%" 
  height="1000px" 
  frameborder="0" 
  allow="fullscreen">
</iframe>
```

### Option 3: Run Locally
```bash
# Clone the repository
git clone https://github.com/YourUsername/ethioputor-amharic.git
cd ethioputor-amharic

# Open in your browser
open index.html
# or double-click index.html in file explorer
```
